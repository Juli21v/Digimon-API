About

Está página trata a cerca de los diferentes juegos, características, y canciones de la franquicia de Digimon, al mismo tiempo representando a partir de imagenes los diferentes tipos de digimon existentes, clasificandolos en nivel y rango del digimon.

Proyecto

En este proyecto se utilizaron, diferentes recursos de HTML, css y javascript para la elaboración de una página funcional, y que con diferentes medios visuales se permite explicar el tema a tratar.
